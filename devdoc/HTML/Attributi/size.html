size: attributo HTML

L'attributo size definisce la larghezza di <input> e l'altezza dell'elemento <select>. Per input, se l' attributo type è text o password , è il numero di caratteri. Deve essere un valore intero 0 o superiore. Se viene specificato no sizeo viene specificato un valore non valido, l'input non ha una dimensione dichiarata e il controllo del modulo sarà la larghezza predefinita in base al programma utente. Se CSS ha come target l'elemento con proprietà che influiscono sulla larghezza, CSS ha la precedenza.

L'attributo size non ha alcun impatto sulla convalida del vincolo.
