Attributi HTML

Gli elementi in HTML hanno attributi ; si tratta di valori aggiuntivi che configurano gli elementi o ne regolano il comportamento in vari modi per soddisfare i criteri desiderati dagli utenti.

Elenco di attributi

Nome attributo Elementi Descrizione
accept <form>,<input> Elenco dei tipi accettati dal server, in genere un tipo di file.
accept-charset <form> Elenco dei set di caratteri supportati.
accesskey Attributi globali Scorciatoia da tastiera per attivare o aggiungere lo stato attivo all'elemento.
allow <iframe> Specifica una policy di funzionalità per l'iframe.
async <script> Specifica una policy di funzionalità per l'iframe.
autocapitalize Attributo globale Imposta se l'input viene automaticamente scritto in maiuscolo quando viene immesso dall'utente
autocomplete <form>,<input>,<select>,<textarea> Indica se i controlli in questo modulo possono, per impostazione predefinita, avere i loro valori completati automaticamente dal browser.
capture <input> Specifica che è possibile acquisire un nuovo file.
challenge <keygen> Una stringa di verifica inviata insieme alla chiave pubblica.
class Attributi globali Spesso utilizzato con CSS per applicare uno stile agli elementi con proprietà comuni.
code <applet> Specifica l'URL del file di classe dell'applet da caricare ed eseguire.
codebase <applet> Questo attributo fornisce l'URL assoluto o relativo della directory in cui sono archiviati i file .class delle applet a cui fa riferimento l'attributo code.
cols <textarea> Definisce il numero di colonne in un'area di testo.
content <meta> Un valore associato http-equiv o name dipendente dal contesto.
contenteditable Attributo globale Indica se il contenuto dell'elemento è modificabile.
coords <area> Un insieme di valori che specificano le coordinate della regione dell'hotspot.
crossorigin <audio>,<img>,<link>,<script>,<video> In che modo l'elemento gestisce le richieste tra origini
data <object> Specifica l'URL della risorsa.
data-* Attributo globale Consente di allegare attributi personalizzati a un elemento HTML.
decoding <img> Indica il metodo preferito per decodificare l'immagine.
default <track> Indica che la traccia deve essere abilitata a meno che le preferenze dell'utente non indichino qualcosa di diverso.
defer <script> Indica che lo script deve essere eseguito dopo che la pagina è stata analizzata.
dir Attributo globale Definisce la direzione del testo. I valori consentiti sono ltr (da sinistra a destra) o rtl (da destra a sinistra)
disabled <button>,<fieldset>,<input>,<keygen>,<optgroup>,<option>,<select>,<textarea> Indica se l'utente può interagire con l'elemento.
draggable Attributo globale Definisce se l'elemento può essere trascinato.
enctype <form> Definisce il tipo di contenuto dei dati del modulo quando method è POST.
for <label>,<output> Descrive gli elementi che appartengono a questo.
hidden Attributi globali Impedisce il rendering di un dato elemento, mantenendo attivi gli elementi figlio, ad esempio gli elementi di script.
high <meter> Indica il limite inferiore dell'intervallo superiore.
http-equiv <meta> Definisce una direttiva pragma.
id Attributo globale Spesso utilizzato con CSS per definire uno stile di un elemento specifico. Il valore di questo attributo deve essere univoco.
integrity <link>,<script> Specifica un valore di integrità della sottorisorsa che consente ai browser di verificare ciò che recuperano.
inputmode <textarea> Fornisce un suggerimento sul tipo di dati che potrebbero essere inseriti dall'utente durante la modifica dell'elemento o del suo contenuto. L'attributo può essere utilizzato con i controlli del modulo (come il valore degli elementi textarea) o negli elementi in un host di modifica (ad esempio, utilizzando l'attributo contenteditable).
ismap <img> Indica che l'immagine fa parte di una mappa immagine lato server.
itemprop Attributo globale
keytype <keygen>
kind <track> Specifica il tipo di traccia di testo.
lang Attributo globale Definisce la lingua utilizzata nell'elemento.
language <script> Definisce il linguaggio di script utilizzato nell'elemento.
list <input> Identifica un elenco di opzioni predefinite da suggerire all'utente.
low <meter> Indica il limite superiore dell'intervallo inferiore.
manifest <html> Specifica l'URL del manifest della cache del documento.
max <input>,<meter>,<progress> Indica il valore massimo consentito.
maxlength <input>,<textarea> Definisce il numero massimo di caratteri consentiti nell'elemento.
minlength <input>,<textarea> Definisce il numero minimo di caratteri consentiti nell'elemento.
method <form> Definisce quale metodo HTTP utilizzare durante l'invio del modulo. Può essere GET(predefinito) o POST.
min <input>,<meter> Indica il valore minimo consentito.
multiple <input>,<select> Indica se è possibile immettere più valori in un input di tipo email o file.
novalidate <form> Questo attributo indica che il modulo non deve essere convalidato al momento dell'invio.
optimum <meter> Indica il valore numerico ottimale.
pattern <input> Definisce un'espressione regolare rispetto alla quale verrà convalidato il valore dell'elemento.
ping <a>,<area> L'attributo ping specifica un elenco di URL separati da spazi da notificare se un utente segue il collegamento ipertestuale.
poster <video> Un URL che indica un fotogramma poster da mostrare fino a quando l'utente non riproduce o cerca.
readonly <input>,<textarea> Indica se l'elemento può essere modificato.
rel <a>,<area>,<link> Specifica la relazione dell'oggetto di destinazione con l'oggetto collegamento.
required <input>,<select>,<textarea> Indica se questo elemento deve essere compilato o meno.
reversed <ol> Indica se l'elenco deve essere visualizzato in ordine decrescente anziché crescente.
role Attributo globale Definisce un ruolo esplicito per un elemento per l'utilizzo da parte delle tecnologie assistive.
rows <textarea> Definisce il numero di righe in un'area di testo.
sandbox <iframe> Impedisce a un documento caricato in un iframe di utilizzare determinate funzioni (come l'invio di moduli o l'apertura di nuove finestre).
scope <th> th definisce le celle a cui si riferisce il test di intestazione (definito nell'elemento).
scoped <style>
selected <option> Definisce un valore che verrà selezionato al caricamento della pagina.
size <input>,<select> Definisce la larghezza dell'elemento (in pixel). Se l'attributo type dell'elemento è text o password allora è il numero di caratteri.
slot Attributo globale Assegna uno slot in un albero ombra DOM shadow a un elemento.
spellcheck Attributo globale Indica se il controllo ortografico è consentito per l'elemento.
srcdoc <iframe>
srclang <track>
start <ol> Definisce il primo numero se diverso da 1.
step <input>
style Attributo globale Definisce gli stili CSS che sovrascriveranno gli stili precedentemente impostati.
summary <table>
tabindex Attributo globale Sostituisce l'ordine di tabulazione predefinito del browser e segue invece quello specificato.
title Attributo globale Testo da visualizzare in una descrizione comando quando si passa il mouse sopra l'elemento.
translate Attributo globale Specificare se i valori degli attributi di un elemento e i valori dei suoi nodi secondari Text devono essere tradotti quando la pagina viene localizzata o se lasciarli invariati.
wrap <textarea> Indica se il testo deve essere avvolto.
