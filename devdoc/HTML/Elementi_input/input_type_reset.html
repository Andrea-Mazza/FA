<input type="reset">

Gli elementi <input type="reset"> vengono visualizzati come pulsanti, con un gestore di eventi click predefinito che reimposta tutti gli input nel modulo ai valori iniziali.

Gli attributi di questo elemento includono gli attributi globali.

Valore

L'attributo value di un elemento <input type="reset"> contiene una stringa che viene utilizzata come etichetta del pulsante. Pulsanti come reset non hanno un valore altrimenti.

Impostazione dell'attributo value

<input type="reset" value="Resetta il form">

Omissione dell'attributo value

Se non specifichi a value, ottieni un pulsante con l'etichetta predefinita (in genere "Ripristina", ma questo varia a seconda dell'agente utente ):

<input type="reset">

Utilizzo di <input type="reset">

In <input type="reset"> i pulsanti vengono utilizzati per ripristinare i moduli. Se vuoi creare un pulsante personalizzato e poi personalizzare il comportamento utilizzando JavaScript, devi utilizzare <input type="button">, o meglio ancora, un elemento <button>.

Un semplice pulsante di ripristino

<form>
  <div>
    <label for="esempio">Inserisci un testo di esempio</label>
    <input id="esempio" type="text">
  </div>
  <div>
    <input type="reset" value="Resetta il form">
  </div>
</form>

Disabilitazione e abilitazione di un pulsante di ripristino

Per disabilitare un pulsante di ripristino, specifica l'attributo globale disabled su di esso, in questo modo:

<input type="reset" value="Disabilitato" disabled>

Nota: vedere la pagina dedicata a <input type="button"> per ulteriori idee sull'abilitazione e la disabilitazione dei pulsanti.

Convalida

I pulsanti non partecipano alla convalida dei vincoli; non hanno alcun valore reale da vincolare.
