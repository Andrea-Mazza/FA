<param>: l'elemento "parametro oggetto"

Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente. Tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento.

L'elemento HTML <param> definisce i parametri per un elemento <object>.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

name

Nome del parametro

value

Specifica il valore del parametro.

type

Utilizzato solo se valuetype è impostato su ref. Specifica il tipo MIME dei valori trovati nell'URI specificato da value.

valuetype

Specifica il tipo valuedell'attributo. I valori possibili sono:

  • data: Valore predefinito. Il valore viene passato all'implementazione dell'oggetto come stringa.
  • ref: il valore è un URI di una risorsa in cui sono archiviati i valori di runtime.
  • object: Un ID di un altro < <object> nello stesso documento.
