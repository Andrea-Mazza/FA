<code>: l'elemento "Codice"

L'elemento HTML <code> mostra il suo contenuto con uno stile che indica che il testo è un breve frammento di codice del computer. Per impostazione predefinita, il testo del contenuto viene visualizzato utilizzando il carattere monospazio predefinito del programma utente.

Gli attributi di questo elemento includono gli attributi globali.

Esempio

Un paragrafo che include <code>

<p>La funzione <code>selectAll()</code> evidenzia tutto il testo
nel campo di immissione in modo che l'utente possa, ad esempio, copiare il testo.</p>

Appunti

Per rappresentare più righe di codice, avvolgi l'elemento <code> all'interno di un elemento <pre>. L'elemento <code> di per sé rappresenta solo una singola frase di codice o riga di codice.

È possibile definire una regola CSS affinché il selettore code sostituisca il carattere predefinito del browser. Le preferenze impostate dall'utente potrebbero avere la precedenza sul CSS specificato.
