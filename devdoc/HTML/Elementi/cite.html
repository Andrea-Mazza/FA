<cite>: l'elemento "Citazione"

L' elemento HTML <cite> viene utilizzato per descrivere un riferimento a un'opera creativa citata e deve includere il titolo di tale opera. Il riferimento può essere in forma abbreviata secondo le convenzioni appropriate al contesto relative ai metadati delle citazioni.

Gli attributi di questo elemento includono gli attributi globali.

Note d'uso

Nel contesto dell'elemento <cite>, un'opera creativa che potrebbe essere citata potrebbe essere, ad esempio, una delle seguenti:

  • Un libro
  • Un documento di ricerca
  • Un tema
  • Un poema
  • Uno spartito musicale
  • Una canzone
  • Una sceneggiatura teatrale o cinematografica
  • Un film
  • Uno spettacolo televisivo
  • Un gioco
  • Una scultura
  • Un dipinto
  • Una produzione teatrale
  • Un gioco
  • Un'opera
  • Un musical
  • Una mostra
  • Un caso legale
  • Un programma per computer
  • Un sito web
  • Una pagina web
  • Un post sul blog o un commento
  • Un post o un commento sul forum
  • Un tweet
  • Un post su Facebook
  • Una dichiarazione scritta o orale
  • E così via.

Vale la pena notare che la specifica del W3C afferma che un riferimento a un'opera creativa, inclusa in un elemento <cite>, può includere il nome dell'autore dell'opera. Tuttavia, la specifica WHATWG per <cite> dice il contrario: che il nome di una persona non deve mai essere incluso, in nessuna circostanza.

Per includere un riferimento alla fonte del materiale citato che è contenuto all'interno di un elemento <blockquote> o <p>, utilizzare sull'elemento l'attributo <cite>.

In genere, i browser impostano il contenuto di un elemento <cite> in corsivo per impostazione predefinita. Per evitare ciò, applica la proprietà CSS font-style all'elemento <cite>.

Esempio

<p>Più informazioni possono essere trovate in <cite> [ISO-0000] </cite>.</p>
