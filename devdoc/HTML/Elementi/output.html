<output>: l'elemento HTML "output"

L'elemento HTML <output> è un elemento contenitore in cui un sito o un'app può inserire i risultati di un calcolo o il risultato di un'azione dell'utente.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

for

Un elenco separato da spazi di altri elementi id, che indica che tali elementi hanno contribuito ai valori di input (o hanno influenzato in altro modo) il calcolo.

form

L'elemento <form> a cui associare l'output ( proprietario del modulo ). Il valore di questo attributo deve essere quello id di un <form> nello stesso documento. (Se questo attributo non è impostato, <output> è associato al suo elemento <form> antenato, se presente.)

Questo attributo ti consente di associare elementi <output> a <form> ovunque nel documento, non solo all'interno di un file <form>. Può anche sovrascrivere un elemento <form> antenato.

name

Il nome dell'elemento.

Il valore <output>, il nome e il contenuto NON vengono inviati durante l'invio del modulo.

Problemi di accessibilità

Molti browser implementano questo elemento come regione aria-live. La tecnologia assistiva annuncerà quindi i risultati delle interazioni dell'interfaccia utente pubblicate al suo interno senza richiedere che l'attenzione venga spostata dai controlli che producono tali risultati.
