<samp>: l'elemento "output di esempio"

L'elemento HTML <samp> viene utilizzato per racchiudere il testo in linea che rappresenta l'output di esempio (o citato) da un programma per computer. I suoi contenuti vengono in genere visualizzati utilizzando il carattere a spaziatura fissa predefinito del browser (come Courier o Lucida Console).

Gli attributi di questo elemento includono gli attributi globali.

Note d'uso

È possibile utilizzare una regola CSS per sovrascrivere il carattere predefinito del browser per l'elemento <samp>; tuttavia, è possibile che le preferenze del browser abbiano la precedenza su qualsiasi CSS specificato.

Il CSS per sovrascrivere il carattere predefinito sarebbe simile a questo:

samp {
          font-family: "Courier";
        }

Nota: se hai bisogno di un elemento che serva da contenitore per l'output generato dal codice JavaScript del tuo sito Web o dell'app, dovresti invece utilizzare l'elemento <output>.

Esempi

<p>Quando il processo è completo, l'utilità emetterà il testo
        <samp>Scansione completata. Hai trovato <em>N</em> risultati.</samp> 
        Puoi quindi procedere al passaggio successivo.</p>
