<acronym>

Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente; tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento

Gli attributi di questo elemento includono gli attributi globali.

Riepilogo

L'elemento HTML <acronym> consente agli autori di indicare chiaramente una sequenza di caratteri che compongono un acronimo o un'abbreviazione di una parola.

Avvertimento: Non usare questo elemento. Usa invece l'elemento <abbr>.

Esempio

<p><acronym title="World Wide Web">WWW</acronym> è solo una componente di Internet. </p>

Stile predefinito

Sebbene lo scopo di questo tag sia puramente per comodità dell'autore, il suo stile predefinito varia da un browser all'altro:

  • Alcuni browser, come Internet Explorer, non hanno uno stile diverso rispetto a un elemento <span>.
  • Opera, Firefox, Chrome e alcuni altri aggiungono una sottolineatura tratteggiata al contenuto dell'elemento.
  • Alcuni browser non solo aggiungono una sottolineatura tratteggiata, ma la mettono anche in maiuscolo; per evitare questo stile, aggiungendo qualcosa come font-variant: none permetterà al CSS di occuparsi di questo caso.

Si consiglia pertanto agli autori web di definire esplicitamente questo elemento o di accettare alcune variazioni tra browser.
