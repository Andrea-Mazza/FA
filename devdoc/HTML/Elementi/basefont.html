Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente; tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento.

L' elemento HTML <basefont> è deprecato. Imposta un carattere, una dimensione e un colore predefiniti per gli altri elementi che discendono dal suo elemento padre. Con questo set, la dimensione del carattere può quindi essere modificata rispetto alla dimensione di base utilizzando l'elemento <font>.

Non dovresti usare questo elemento; invece, dovresti usare le proprietà CSS come font, font-family, font-size, e color per cambiare la configurazione del carattere per un elemento e il suo contenuto.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

color

Questo attributo imposta il colore del testo utilizzando un colore denominato o un colore specificato nel formato esadecimale #RRGGBB.

face
Questo attributo contiene un elenco di uno o più nomi di caratteri. Il testo del documento nello stile predefinito viene visualizzato nel primo tipo di carattere supportato dal browser del client. Se nessun font elencato è installato sul sistema locale, il browser in genere utilizza per impostazione predefinita il font proporzionale o a larghezza fissa per quel sistema.
size
Questo attributo specifica la dimensione del carattere come valore numerico o relativo. I valori numerici vanno da 1 a 7, dove 1 è il più piccolo e 3 quello predefinito.

Note sull'utilizzo

Non utilizzare questo elemento! Sebbene una volta (imprecisamente) normalizzato in HTML 3.2, non era supportato in tutti i principali browser. Inoltre, i browser, e anche le successive versioni di browser, non l'hanno mai implementato allo stesso modo: in pratica, il suo utilizzo ha sempre portato a risultati indeterminati

L'elemento <basefont> è stato deprecato nello standard contemporaneamente a tutti gli elementi relativi solo allo stile. A partire da HTML 4, HTML non trasmette più informazioni sullo stile (al di fuori dell'elemento <style> o dell'attributo style di ciascun elemento). In HTML5, questo elemento è stato completamente rimosso. Per qualsiasi nuovo sviluppo web, lo stile dovrebbe essere scritto usando solo CSS

Il precedente comportamento dell'elemento <font> può essere ottenuto e controllato ancora meglio utilizzando le proprietà CSS Fonts .

Appunti

  • HTML 3.2 supporta l'elemento basefont ma solo con l'attributo size.
  • Le rigide specifiche HTML e XHTML non supportano questo elemento.
  • Nonostante facciano parte degli standard di transizione, alcuni browser incentrati sugli standard come Mozilla e Opera non supportano questo elemento.
  • Questo elemento può essere imitato con una regola CSS sull'elemento <body>.
  • XHTML 1.0 richiede una barra finale per questo elemento: <basefont />.
