<frame>

Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente. Tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento.

L' elemento HTML <frame> definisce un'area particolare in cui può essere visualizzato un altro documento HTML. Un frame dovrebbe essere utilizzato all'interno di un <frameset>.

L'utilizzo dell'elemento <frame> non è incoraggiato a causa di alcuni svantaggi come problemi di prestazioni e mancanza di accessibilità per gli utenti con lettori di schermo. Invece dell'elemento <frame>, può essere preferito <iframe.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

src

Questo attributo specifica il documento che verrà visualizzato dal frame.

name

Questo attributo viene utilizzato per etichettare i frame. Senza etichettatura, ogni collegamento si aprirà nel frame in cui si trova, il frame padre più vicino. Vedere l'attributo target per ulteriori informazioni.

noresize

Questo attributo impedisce il ridimensionamento dei frame da parte degli utenti.

scrolling

Questo attributo definisce l'esistenza di una barra di scorrimento. Se questo attributo non viene utilizzato, il browser aggiunge una barra di scorrimento quando necessario. Ci sono due scelte: "yes" per forzare una barra di scorrimento anche quando non è necessaria e "no" per forzare alcua barra di scorrimento anche quando è necessaria.

marginheight

Questo attributo definisce l'altezza del margine tra i fotogrammi.

marginwidth

Questo attributo definisce la larghezza del margine tra i fotogrammi.

frameborder

Questo attributo consente di specificare il bordo di una cornice.

Esempio

<frameset cols="50%, 50%">
          <frame src="indirizzoDaIncludereNelFrame.com" />
          <frame src="indirizzoDaIncludereNellIframe.com" />
        </frameset>
