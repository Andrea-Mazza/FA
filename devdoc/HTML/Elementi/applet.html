<applet>: l'elemento Embed Java Applet

Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente. Tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento.

L'elemento obsoleto Applet HTML (<applet>) incorpora un'applet Java nel documento; questo elemento è stato deprecato a favore di <object>.

L'uso di applet Java sul Web è deprecato; la maggior parte dei browser non supporta più l'uso dei plug-in, incluso il plug-in Java.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

align

Questo attributo viene utilizzato per posizionare l'applet sulla pagina rispetto al contenuto che potrebbe scorrere attorno ad essa. La specifica HTML 4.01 definisce i valori di <bottom>, <left>, <middle>, <right> e <top>, mentre Microsoft e Netscape potrebbero anche supportare <absbottom>, <absmiddle>, <baseline>, <center> e <texttop>.

alt

Questo attributo fa sì che un testo descrittivo alternativo venga visualizzato sui browser che non supportano Java. I designer di pagine dovrebbero anche ricordare che il contenuto racchiuso all'interno dell'elemento <applet> può anche essere visualizzato come testo alternativo.

archive

Questo attributo si riferisce a una versione archiviata o compressa dell'applet e ai file di classe associati, che potrebbero aiutare a ridurre il tempo di download.

code

Questo attributo specifica l'URL del file di classe dell'applet da caricare ed eseguire. I nomi dei file dell'applet sono identificati da un'estensione del nome file .class. L'URL specificato dal codice potrebbe essere relativo all'attributo <codebase>.

codebase

Questo attributo fornisce l'URL assoluto o relativo della directory in cui sono archiviati i file .class delle applet a cui fa riferimento l'attributo code.

datafld

Questo attributo, supportato da Internet Explorer 4 e versioni successive, specifica il nome della colonna dall'oggetto origine dati che fornisce i dati associati. Questo attributo può essere utilizzato per specificare i vari elementi <param> passati all'applet Java.

datasrc

Come <datafld>, questo attributo viene utilizzato per l'associazione dati in Internet Explorer 4. Indica l'ID dell'oggetto origine dati che fornisce i dati associati agli elementi <param> associati all'applet.

height

Questo attributo specifica l'altezza, in pixel, necessaria all'applet.

hspace

Questo attributo specifica uno spazio orizzontale aggiuntivo, in pixel, da riservare su entrambi i lati dell'applet.

mayscript

Nell'implementazione di Netscape, questo attributo consente l'accesso a un'applet da parte di programmi in un linguaggio di scripting incorporato nel documento.

name

Questo attributo assegna un nome all'applet in modo che possa essere identificato da altre risorse; in particolare script.

object

Questo attributo specifica l'URL di una rappresentazione serializzata di un'applet.

src

Come definito per Internet Explorer 4 e versioni successive, questo attributo specifica un URL per un file associato per l'applet. Il significato e l'uso non sono chiari e non fanno parte dello standard HTML.

vspace

Questo attributo specifica uno spazio verticale aggiuntivo, in pixel, da riservare sopra e sotto l'applet.

width

Questo attributo specifica in pixel la larghezza necessaria all'applet.
