<hr>: l'elemento "Pausa tematica"

L'elemento HTML <hr> rappresenta un'interruzione tematica tra gli elementi a livello di paragrafo: ad esempio, un cambio di scena in una storia o uno spostamento di argomento all'interno di una sezione.

Storicamente, questo è stato presentato come una regola o una linea orizzontale. Sebbene possa ancora essere visualizzato come regola orizzontale nei browser visuali, questo elemento è ora definito in termini semantici, piuttosto che in termini di presentazione, quindi se desideri disegnare una linea orizzontale, dovresti farlo usando il CSS appropriato.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

align

Imposta l'allineamento della regola sulla pagina. Se non viene specificato alcun valore, il valore predefinito è left.

color

Imposta il colore della regola tramite il nome del colore o il valore esadecimale.

noshade

Imposta la regola per non avere ombreggiatura.

size

Imposta l'altezza, in pixel, della regola.

width

Imposta la lunghezza della regola sulla pagina tramite un pixel o un valore percentuale.

Esempio

HTML

<p>
  Questo è il primo paragrafo
  Questo è il primo paragrafo
  Questo è il primo paragrafo
  Questo è il primo paragrafo                                
</p>

<hr>
