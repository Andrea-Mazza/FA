<font>

Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente. Tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento.

Sommario

L' elemento HTML <font> definisce la dimensione del carattere, il colore e la faccia per il suo contenuto.

Avvertimento: non utilizzare questo elemento. Usa le proprietà CSS Fonts per dare uno stile al testo.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

color

Questo attributo imposta il colore del testo utilizzando un colore denominato o un colore specificato nel formato esadecimale #RRGGBB.

face

Questo attributo contiene un elenco separato da virgole di uno o più nomi di font. Il testo del documento nello stile predefinito viene visualizzato nel primo tipo di carattere supportato dal browser del client. Se nessun font elencato è installato sul sistema locale, il browser in genere utilizza per impostazione predefinita il font proporzionale o a larghezza fissa per quel sistema.

size

Questo attributo specifica la dimensione del carattere come valore numerico o relativo. I valori numerici vanno da 1 a 7 con il valore più piccolo 1 e quello predefinito 3. Può essere definito utilizzando un valore relativo, come +2 o -3, che lo imposta rispetto al valore dell'attributo size dell'elemento <basefont>, o relativo a 3, il valore predefinito, se non esiste.
