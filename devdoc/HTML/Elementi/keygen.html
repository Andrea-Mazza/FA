<keygen>

Deprecato: questa funzione non è più consigliata. Sebbene alcuni browser possano ancora supportarlo, potrebbe essere già stato rimosso dagli standard Web pertinenti, potrebbe essere in fase di eliminazione o potrebbe essere conservato solo per motivi di compatibilità. Evita di usarlo e, se possibile, aggiorna il codice esistente; tieni presente che questa funzione potrebbe cessare di funzionare in qualsiasi momento.

L' elemento HTML <keygen> esiste per facilitare la generazione del materiale della chiave e l'invio della chiave pubblica come parte di un modulo HTML . Questo meccanismo è progettato per essere utilizzato con i sistemi di gestione dei certificati basati sul Web. Si prevede che l' elemento <keygen> verrà utilizzato in un modulo HTML insieme ad altre informazioni necessarie per creare una richiesta di certificato e che il risultato del processo sarà un certificato firmato.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

autofocus

Questo attributo booleano consente di specificare che il controllo deve avere lo stato attivo per l'input quando la pagina viene caricata, a meno che l'utente non lo sovrascriva, ad esempio digitando un controllo diverso. Solo un elemento del modulo in un documento può avere l'attributo autofocus, che è un booleano.

challenge

Una stringa di verifica inviata insieme alla chiave pubblica. Il valore predefinito è una stringa vuota se non specificato.

disabled

Questo attributo booleano indica che il controllo del modulo non è disponibile per l'interazione.

form

L'elemento del modulo a cui questo elemento è associato (il relativo proprietario del modulo ). Il valore dell'attributo deve essere un id di un elemento <form> nello stesso documento. Se questo attributo non è specificato, questo elemento deve essere un discendente di un elemento <form>. Questo attributo ti consente di posizionare elementi <keygen> ovunque all'interno di un documento, non solo come discendenti dei loro elementi del modulo.

keytype

Il tipo di chiave generata. Il valore predefinito è RSA.

name

Il nome del controllo, che viene inviato con i dati del modulo.
