<address>: l'elemento di Indirizzo di contatto

L'elemento HTML <address> che l'allegato HTML fornisce informazioni di contatto per una o più persone o per un'organizzazione.

Le informazioni di contatto forniti dai contenuti di un elemento <address> possono assumere qualsiasi forma appropriata al contesto e possono includere qualsiasi tipo di informazioni di contatto necessarie, come un indirizzo fisico, URL, indirizzo e-mail, numero di telefono, social media handle, coordinate geografiche, e così via. L'elemento <address> deve includere il nome della persona, delle persone o dell'organizzazione a cui si riferiscono le informazioni di contatto.

<address> può essere utilizzato in una varietà di contesti, ad esempio fornendo le informazioni di contatto di un'azienda nell'intestazione della pagina o indicando l'autore di un articolo includendo un elemento <address> all'interno di <article>.

Gli attributi di questo elemento includono gli attributi globali.

Note sull'utilizzo

  • L'elemento <address> può essere utilizzato solo per rappresentare le informazioni di contatto per il suo più vicino predecessore <article> o <body>.
  • Questo elemento non deve contenere più informazioni delle informazioni di contatto, come una data di pubblicazione (che appartiene ad un elemento <time>).
  • Tipicamente un elemento <address> può essere posizionato all'interno dell'elemento <footer> della sezione corrente, se presente

Esempi

Questo esempio dimostra l'uso di <address> per delimitare le informazioni di contatto per l'autore di un articolo.

<address>
    Puoi contattare l'autore a <a
    href="http://dominio.com/contatti">
    www.dominio.com </a>.
</address>
