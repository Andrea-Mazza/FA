<map>: l'elemento "Mappa-immagine"

L' elemento HTML <map> viene utilizzato con gli elementi <area> per definire una mappa immagine (un'area di collegamento cliccabile).

Attributi

Gli attributi di questo elemento includono gli attributi globali.

name

L'attributo name assegna alla mappa un nome in modo che possa essere referenziata. L'attributo deve essere presente e deve avere un valore non vuoto senza caratteri spaziali. Il valore dell'attributo name non deve essere uguale al valore dell'attributo name di un altro elemento <map> nello stesso documento. Se idviene specificato anche l'attributo, entrambi gli attributi devono avere lo stesso valore.
