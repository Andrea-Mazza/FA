<blockquote>: l'elemento 'blockquote' (tradotto: virgolette bloccate)

L'elemento HTML <blockquote> indica che il testo allegato è una citazione estesa. Di solito, questo è reso visivamente da un'indentazione. Un URL per la fonte della citazione può essere fornito usando l'attributo cite, mentre una rappresentazione testuale della fonte può essere fornita usando l'elemento <cite>.

Attributi

Gli attributi di questo elemento includono gli attributi globali.

cite
Un URL che designa un documento di origine o un messaggio per le informazioni citate. Questo attributo ha lo scopo di puntare a informazioni che spiegano il contesto o il riferimento per la citazione.

Note d'uso

Per modificare il rientro applicato al testo citato, utilizzare le proprietà CSS margin-left e/o margin-right, o la proprietà abbreviata margin.

Per includere virgolette più brevi in ​​linea anziché in un blocco separato, utilizzare l'elemento <q> (Quotation).

Esempi

<blockquote cite="https://it.wikipedia.org/wiki/HTML">
  <p>In informatica l'HyperText Markup Language 
 (traduzione letterale: linguaggio a marcatori per ipertesti), 
 comunemente noto con l'acronimo HTML, è un linguaggio di markup. 
 Nato per la formattazione e impaginazione di 
 documenti ipertestuali disponibili nel web 1.0, 
 oggi è utilizzato principalmente per il disaccoppiamento 
 della struttura logica di una pagina web 
 (definita appunto dal markup) e la sua rappresentazione, 
 gestita tramite gli stili CSS  per adattarsi alle nuove esigenze 
 di comunicazione e pubblicazione all'interno di Internet.</p>
</blockquote>
